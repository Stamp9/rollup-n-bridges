## Getting Started

### Prerequisites

- Node.js 20+.
- pnpm (bundled with Node) or a compatible package manager.

### Installation

#### Backend

1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Start the indexer:
   ```bash
   pnpm indexter
   ```
#### Frontend

1. In a separate terminal, run:
   ```bash
   npm run dev
   ```
2. Open `http://localhost:5173` in your browser.
